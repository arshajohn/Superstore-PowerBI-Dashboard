# Superstore Sales & Profit Dashboard

## Project Overview
This project analyzes Superstore sales data using Power BI.

## Features
- Sales Analysis
- Profit Analysis
- Regional Performance
- Product Insights
- Interactive Filters

## Tools Used
- Power BI
- DAX
- Power Query
